FAQ

Does it work with Israeli stocks (TASE)? ▼

Yes. Add .TA suffix (e.g., TEVA.TA, NICE.TA). Uses Jerusalem timezone for TASE hours (Sun–Thu 9:59–17:25).

How do I run it? ▼

Clone the repo → pip install -r requirements.txt → add API keys to .env → streamlit run trade_app.py

What API keys do I need? ▼

Marketaux (news, free 100 req/day). Optionally: Finnhub, Reddit PRAW, Brave Search. Market data via Yahoo Finance — no key needed.

Is my data saved anywhere? ▼

100% local. Watchlist and journal saved as JSON files on your machine. Nothing sent to any server.
